# AstrBot 简单的轮盘赌

群聊多人轮盘赌插件，默认支持 2 到 6人，也可通过配置调高人数上限；包含弹仓轮、公开道具和房主处决挂机玩家。
普通模式从 `v1.0.0` 开始和异界战争拆分，复杂扩展玩法请使用 `astrbot-plugin-roulette-isekai`。

## 指令

| 指令 | 说明 |
| --- | --- |
| `/轮盘创建` | 创建普通轮盘房间 |
| `/轮盘加入` | 加入等待中的房间 |
| `/退出房间` | 游戏开始前退出等待房间 |
| `/轮盘开始` | 房主或超级管理员开始游戏 |
| `/开自己` | 对自己开枪 |
| `/开 @用户` | 对指定存活玩家开枪 |
| `/听天由命` | 多人局随机选择存活玩家作为目标 |
| `/使用道具 道具名` | 使用道具 |
| `/使用顺手牵羊 @用户` | 道具短指令，其他道具同理 |
| `/梭哈` | 查看梭哈确认提示，不直接执行 |
| `/梭哈 数量` | 最多使用指定数量，不跨弹仓 |
| `/梭哈 all` | 梭哈当前弹仓剩余全部子弹 |
| `/查看道具` | 查看自己的道具 |
| `/道具帮助 道具名` | 查看指定道具说明 |
| `/轮盘状态` | 查看当前房间状态 |
| `/死亡榜` | 查看本群死亡次数排行 |
| `/轮盘处决 @用户` | 房主或超级管理员处决挂机玩家 |
| `/轮盘结束` | 房主或超级管理员结束当前房间 |
| `/轮盘帮助` | 查看帮助 |


## 基础规则

| 人数 | 血量 | 道具开始 | 每次发放 | 背包上限 | 前期弹仓限制 |
| --- | ---: | ---: | --- | ---: | --- |
| 2 人 | 5 | 第 3 轮 | 每人随机 2 个 | 6 | 前 2 轮最多 4 发 |
| 3-4 人 | 3 | 第 3 轮 | 每人随机 2 个 | 4 | 前 2 轮最多 6 发 |
| 5人及以上 | 2 | 第 2 轮 | 每人 1 个 | 3 | 开局即最多 10 发 |
- 弹仓每轮 3 到 10 发，至少 1 发实弹、1 发空弹；公开数量，不公开顺序。
- 道具只能在自己的行动回合使用。被跳过回合时不能使用道具、不能开枪。
- 房主或超级管理员可以处决挂机玩家，插件不做自动倒计时。
- 对玩家造成伤害后会提示剩余生命。
- 轮到玩家行动时会 @ 提醒。
- `/轮盘状态` 不显示弹仓剩余信息，以保留危机感。
- `/梭哈` 只显示确认提示，不直接执行。
- `/梭哈 数量` 可以指定最多使用几发，遇到实弹会提前停止。
- `/梭哈 all` 会使用当前弹仓剩余全部子弹。
- 梭哈执行过程不会跨弹仓；如果全部为空弹并打空当前弹仓，会进入下一弹仓轮且仍由当前玩家继续行动。

## 道具

| 道具 | 效果 |
| --- | --- |
| 魔镜 | 公开查看当前子弹是真弹还是空弹 |
| 止痛片 | 回复 1 血，不超过血量上限 |
| 肘击 | 退出当前子弹并公开类型 |
| 改装工具 | 下一枪若为实弹则伤害 +1 |
| 束线带 | 指定一名存活玩家跳过下次行动 |
| 反转器 | 反转当前子弹，实弹变空弹，空弹变实弹 |
| 测弹仪 | 随机显示弹仓中某一位置的子弹类型 |
| 顺手牵羊 | 随机偷取目标 1 个普通道具并立即使用 |
| 怪味蘑菇 | 50% 回复 2 血，50% 受到 1 点伤害 |

## v1.0.0 更新

- 普通模式与异界战争拆分，异界战争迁移到 `astrbot-plugin-roulette-isekai`。
- 新增 `/道具帮助 道具名`。
- 新增 WebUI 分人数规则配置。
- 道具发放数量支持固定值，例如 `3`，也支持范围，例如 `1-3`。

## WebUI 配置

可在 AstrBot WebUI 中调整模式的分人数规则：

- 初始血量
- 第几轮开始发放道具
- 每次发放道具数量，支持 `3` 或 `1-3`
- 背包上限
- 前几轮限制子弹数
- 前期最多几发子弹


| 配置项 | 类型 | 默认值 | 说明 |
| --- | --- | --- | --- |
| `whitelist_groups` | list | `[]` | 群聊白名单；填写后仅这些群可用 |
| `blacklist_groups` | list | `[]` | 群聊黑名单 |
| `max_players` | int | `6` | 最大玩家数。默认 6，可调高到 7 人或以上，但超过 6 人后平衡性与回合节奏可能下降 |
| `use_player_codes` | bool | `true` | 是否使用 `P1`、`P2` 等玩家代号 |
| `player_name_max_length` | int | `8` | 玩家昵称截断长度 |
